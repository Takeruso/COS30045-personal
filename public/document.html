Data Sources

OECD Data Explorer - Freshwater Resources

This dataset provides statistical data related to "Freshwater Resources" compiled by the OECD. It includes key indicators such as precipitation resources, groundwater resources, and renewable water resources. The data is structured by country and time period.

OECD Data Explorer - Freshwater Abstractions

This dataset provides information on "Freshwater Abstractions," including abstraction rates by sector such as irrigation, industry, domestic use, and total abstraction. It includes data by country and time period.

Custom GeoJSON

A custom GeoJSON file provides geographic boundaries for mapping and visualising data. It contains geographic features such as administrative regions and their spatial coordinates, essential for generating choropleth maps or other geographic visualisations.
